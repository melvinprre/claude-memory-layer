# Log

Journal append-only des opérations sur le vault. Une opération = une entrée. Append-only strict.

Format : `## [YYYY-MM-DD] type | titre`

Types valides : `init`, `remember`, `decision`, `recap`, `methodize`, `session`, `refactor`, `capture`.

---

## [{{ YYYY-MM-DD }}] init | memory layer
- Création initiale du vault depuis le template `claude-memory-layer`
